## Get Started

#### 1. Clone the Repo

On the command prompt run the following commands
```sh
$ git clone https://github.com/SachinMasai/CodeFetch.git

$ cd saas-app/

$ npm install

$ npm install -g json-server

```

#### 2. Run the server

Run the following command

```sh
$ npm run dev
```
